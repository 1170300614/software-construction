JUnit Test organization
7.5 Testing and Test-First Programming
▪ Have a test class FooTest for each public
class Foo
▪ Have a source directory and a test
directory
– Store FooTest and Foo in the same package
– Tests can access members with default
(package) visibility
▪ 注意：需要将src目录和test目录均指定为源代
码目录(source folders)
