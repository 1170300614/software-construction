JUnit
7.5 Testing and Test-First Programming
▪ setUpBeforeClass or @BeforeClass or @BeforeAll （JUnit3/4/5）
– run once before any of the test methods in the class, usually for initialization
@BeforeClass
public static void setUpBeforeClass() throws Exception {
System.out.println("this is before class");
}
▪ tearDownAfterClass or @AfterClass or @AfterAll （JUnit3/4/5）
– run after all the tests in the class have been run, usually for clean-up
@AfterClass
public static void tearDownAfterClass() throws Exception {
System.out.println("this is after class");
}
