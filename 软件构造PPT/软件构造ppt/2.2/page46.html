2.2 Process, Systems, and Tools of Software Construction
Compiled Languages
▪ Version-control tool
▪ Source trees and object trees: The set of source files and compiled
object files that a particular developer works with.
▪ Compilation tools: The tools that take input files and generate
output files (for example, converting source code files into object
code and executable programs). Common examples of compilation
tools include a C or Java compiler, but they also include
documentation and unit test generators.
▪ Build machines: The computing equipment on which the
compilation tools are executed.
▪ Release packaging and target machines: The method by which the
software is packaged, distributed to end users, and then installed on
the target machine.
