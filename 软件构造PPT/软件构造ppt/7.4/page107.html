Debugger
7.4 Debugging
▪ Debuggers are software tools which enable the programmer to
monitor the execution of a program, stop it, restart it,
set breakpoints, and change values in memory.
▪ Main Debugger Operations
– Stepping Through the Source Code
• Breakpoints
• Single-stepping
• Resume operation
• Temporary breakpoints
– Inspecting Variables
– Issuing an “All Points Bulletin” for Changes to a Variable
• A watchpoint combines the notions of breakpoint and variable inspection.
– Moving Up and Down the Call Stack
