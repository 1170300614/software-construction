6.3 Maintainability-Oriented Construction Techniques
Memento Pattern
class Caretaker {
private List<Memento> mementos
= new ArrayList<>();
public void addMemento(Memento m) {
mementos.add(m);
}
public Memento getMemento(int m) {
return mementos.get(m);
}
}
