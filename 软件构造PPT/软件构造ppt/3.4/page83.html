Generics in Java
3.4 Object-Oriented Programming (OOP)
▪ A type variable(类型变量) is an unqualified(无限制的) identifier.
– They are introduced by generic class declarations, generic interface
declarations, generic method declarations
▪ A class is generic if it declares one or more type variables. 类中如果
声明了一个或多个泛型变量，则为泛型类
– These type variables are known as the type parameters(类型参数) of the
class. 这些类型变量称为类的类型参数
– It defines one or more type variables that act as parameters.
– A generic class declaration defines a set of parameterized types, one for
each possible invocation of the type parameter section.
– All of these parameterized types share the same class at runtime.
