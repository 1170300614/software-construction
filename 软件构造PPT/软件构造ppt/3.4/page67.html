Overloading
3.4 Object-Oriented Programming (OOP)
▪ Function overloading is the ability to create multiple methods of the
same name with different implementations.
– Calls to an overloaded function will run a specific implementation of that
function appropriate to the context of the call, allowing one function call
to perform different tasks depending on context. 可以根据需要选择方法的
实现
▪ Overloading is a static polymorphism
– A function call is resolved using the ‘best match technique’, i.e. the
function is resolved depending upon the argument list. 静态的多态，通过
参数列表决定依赖哪个实现
– Static type checking in function calls
– The determination of which of these methods are used is resolved at
compile time. 在编译时决定调用哪个方法
