Subtypes
3.4 Object-Oriented Programming (OOP)
▪ A type is a set of values. 一个类型代表了一组值
– The Java List type is defined by an interface. If we think about all possible
List values, none of them are List objects: we cannot create instances of an
interface.
– Instead, those values are all ArrayList objects, or LinkedList objects, or
objects of another class that implements List .
▪ A subtype is simply a subset of the supertype 子类型是超类型的子集
– ArrayList and LinkedList are subtypes of List .
