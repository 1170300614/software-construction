3.4 Object-Oriented Programming (OOP)
Using static factory instead of constructor
▪ Hiding the implementation completely is a tradeoff, because
sometimes the client wants a choice of implementations with
different characteristics. 是否完全隐藏实现需要进行权衡
▪ That’s why ArrayList and LinkedList are exposed by the Java
library, because they vary in performance of operations like get()
and insert().
