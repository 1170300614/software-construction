Dynamic dispatch
3.4 Object-Oriented Programming (OOP)
dividend.divide(divisor) # dividend / divisor
▪ This is thought of as sending a message named divide with
parameter divisor to dividend.
▪ An implementation will be chosen based only on dividend's type
(perhaps rational, floating point, matrix), disregarding the type or
value of divisor.
(基于dividend的类型选择具体实现，而不考虑divisor的值和类型)
