3.4 Object-Oriented Programming (OOP)
Tips for overriding a method
▪ If you want to override a method:
– Make sure signatures match 签名保持一致
– Use @Override so compiler has your back(compiler 会检查覆盖方法和被
覆盖的方法签名是否完全一致)
– Do copy-and-paste declarations (or let IDE do it for you)
– The visibility can keep the same or increased, but not reduced.
class Book{
protected int getPrice(){
return 30;
}
}
public class ComputerBook extends Book{
int getPrice(){ return 10 } //error
}
