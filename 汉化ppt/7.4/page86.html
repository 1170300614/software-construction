Print debugging/tracing
7.4 Debugging
▪ Print debugging (or tracing) is the act of watching (live or
recorded) trace statements, or print statements, that indicate the
flow of execution of a process. This is sometimes called printf
debugging, due to the use of the printf function in C.
▪ It is to add trace code to the program to print out values of
variables as the program executes, using printf() or
System.out.println() statements, for example. It’s Simple but
effective.
System.out.println("Entering callMethod");
result = callMethod();
System.out.println("The result is " + result);
