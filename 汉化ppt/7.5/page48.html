JUnit
7.5 Testing and Test-First Programming
▪ setUp or @Before or @BeforeEach （JUnit3/4/5）
– @Before causes that method to be run before each Test method
@Before
public void setUp() throws Exception {
System.out.println("this is before");
}
▪ tearDown or @After or @AfterEach （JUnit3/4/5）
– @After causes that method to be run after each Test method.
@After
public void tearDown() throws Exception {
System.out.println("this is Down");
}
