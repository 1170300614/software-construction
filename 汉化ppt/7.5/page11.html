Testing levels
7.5 Testing and Test-First Programming
▪ Unit testing(单元测试): refers to tests that verify the functionality of
a specific section of code, usually at the function level (in OO: class
level).
▪ Integration testing(集成测试): the combined execution of two or
more classes, packages, components, subsystems that have been
created by multiple programmers or programming teams.
▪ System testing(系统测试): to test a completely integrated system to
verify that the system meets its requirements, which executes the
software in its final configuration.
