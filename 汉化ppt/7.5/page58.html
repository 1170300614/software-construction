7.5 Testing and Test-First Programming
Guidelines for Equivalence Partitioning
▪ Equivalence classes may be defined according to the following
guidelines:
– If an input condition specifies a range, one valid and two invalid
equivalence classes are defined. 输入是一个范围，则定义1个有效和2个无
效等价类
– If an input condition requires a specific value, one valid and two invalid
equivalence classes are defined. 输入是特定值，则定义1个有效和2个无效
等价类
– If an input condition specifies a member of a set, one valid and one invalid
equivalence class are defined. 输入是集合的成员，则定义1个有效和1个无
效等价类
– If an input condition is Boolean, one valid and one invalid class are
defined. 输入是布尔型，则定义1个有效和1个无效等价类。
