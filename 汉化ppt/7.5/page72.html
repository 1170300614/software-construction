7.5 Testing and Test-First Programming
▪ Now let’s pick test values that cover all these classes:
– (1, 2) covers a < b, a > 0, b > 0
– (-1, -3) covers a > b, a < 0, b < 0
– (0, 0) covers a = b, a = 0, b = 0
– (Integer.MIN_VALUE, Integer.MAX_VALUE) covers a < b, a = minint,
b = maxint
– (Integer.MAX_VALUE, Integer.MIN_VALUE) covers a > b, a = maxint,
b = minint
