Encapsulate Field
9.2 Refactoring Techniques and Tool Support
There is a public field.
Make it private and provide accessors.
▪ Un-encapsulated data is a no-no in OO application design.
▪ Use property get and set procedures to provide public access to
private (encapsulated) member variables.
public class Course {
public List students;
}
int classSize =
course.students.size();
public class Course {
private List students;
public List getStudents() {
return students;
}
public void setStudents(List s) {
students = s;
}
}
int classSize =
course.getStudents().size();
